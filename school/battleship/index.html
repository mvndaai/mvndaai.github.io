Setup

  • User places their ships
  • When all ships are placed press "Start"

Gameplay

  • This is a turn base gamed.
  • User chooses a space to attack on the opponent's field.
  • If attack is a hit, you get one icon, if a miss a different one.
  • After the user's turn opponent attacks with the same results on users map.
  • When all the ships of one player are destroyed, that player is defeated.


Preliminary Software Design (example)

Objects(vars in list):

Game

  • boardSize: int of lenght/width of the board
  • ships: int of number of ships
  • players: array of both player
  • userTurn: boolean to choose whose turn it is
  • Ex. {boardSize:8, ships:3, players:[], userTurn:true}

Player

  • name: name of player
  • ships: array of ship objects
  • attacks: array of attack locations Ex: x.y
  • hits: array of hit locations Ex: x.y
  • shipVisibility: bool if the ships are displayed on the board
  • opponantName: name of the opponant
  • boardDiv: the html element the board updates to
  • Ex. {name:"user", ships:[], attacks:["1.1","2.5"], hits:["2.5"], shipVisibility:true, opponantName:"comp", boardDiv:""}

Ship

  • head: int array x,y location on the grid to head of ship
  • orientation: enum: horizontal or vertical
  • length: int lenght of ship
  • Ex. {head:"5.1", orientation:"horizontal", length:5}

Functions/Methods:

  • buildBoard() { Creates the board depending on size and ships }
  • setupMode() { Lets the user place their peices on the boards and waits till start button is clicked }
  • playMode() { Goes back and forth between players till the game ends }
  • endMode() { The game is over and it shows the end }
  • placeShips() { Lets the user place thier ships in setupMode }
  • userMove() { User chooses a place on the opponent map and calls checkHit }
  • computerMove() { Computer chooses a place on the opponent map and calls checkHit }
  • checkHit() { Checks if a attack hit a ship and updates ships and map if checks endGame }
  • updateShipCount() { Displays how many hits are left till someone wins }
  • endGame() { Checks if a player is out of ships, if so it goes to end mode }
